# Murmur marketing site

Static landing page for **Murmur**, a privacy-first, accessibility-grade macOS dictation app.

## Files
- `index.html` - the landing page (single file, no build step).
- `home-screen.html` - the in-app home screen mockup, embedded as the hero product preview.
- `.nojekyll` - tells GitHub Pages to serve the files as-is (skip Jekyll).

## Deploy (GitHub Pages)
In the repo settings, enable **Pages** with:
- Source: **Deploy from a branch**
- Branch: **main** / **root**

The site is then served at `https://datnntqn.github.io/murmur-mac/`.

## Notes
Design: light Spectrum brand, magenta (`#D81B83`) accent, rainbow reserved for the logo and the
voice waveform. Copy is verified against the app implementation; competitor pricing and the
quoted testimonial should be re-checked and properly cited before any public launch.
